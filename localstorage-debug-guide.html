LocalStorage デバッグガイド

1. ブラウザ開発者ツールでの確認方法

各ページでlocalStorageの内容を確認する手順：

  1. 対象のページ（index.html、editor.html、test-localstorage.html）を開く
  2. F12キーまたは右クリック→「検証」で開発者ツールを開く
  3. 「Application」タブ（Chromeの場合）または「Storage」タブ（Firefoxの場合）を選択
  4. 左側のメニューから「Local Storage」→「file://」を選択
  5. 右側にlocalStorageの内容が表示される
// コンソールで直接確認する場合 console.log(localStorage.getItem('mapDrivingAssets')); console.log(JSON.parse(localStorage.getItem('mapDrivingAssets')));

4. file:// プロトコルの制限事項

⚠️ 重要な注意事項

file:// プロトコルを使用している場合、ブラウザによってlocalStorageの動作が異なる場合があります：

  • Chrome: file:// でもlocalStorageは正常に動作しますが、同じディレクトリのファイル間でのみ共有されます
  • Firefox: file:// でlocalStorageが制限される場合があります
  • Safari: file:// でのlocalStorageに制限があります

✅ 推奨される解決方法

ローカルサーバーを使用してテストすることを推奨します：

# Python 3の場合 python -m http.server 8000 # Node.jsの場合（http-serverをインストール済み） http-server -p 8000 # その後、ブラウザで以下にアクセス http://localhost:8000/index.html
